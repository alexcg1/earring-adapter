# Earring Adapter

This model allows you to connect standard hook-style earrings to your ears using magnets. There are two different versions:

- **With stem**: The earring dangles on a short stem below the magnet
- **No stem**: The earring sits directly atop the magnet

The goal to be able to wear standard earrings without piercing your ears.

## You will need

- A way to produce the adapter, e.g. 3D printer, CNC
- For each earring you want to adapt:
  * The earring itself
  * Two small magnets
  * Glue (optional, but recommended for stemless version)

## Instructions

Depending on which model you want, print the file and insert your magnet into the magnet cavity. (For the model without stem, do this _after_ inserting the earring hook). Ideally the magnet should have a tight fit. You can optionally secure it with glue.

For the model with the stem, attach earring hook through the smaller hole. For the model with no stem, you'll need to cut the earring hook to fit the hook hole, then glue in place to secure it.

Attach the adapter to the front of your earlobe by using another magnet on the back of your earlobe.

Dimensions can be altered in the sheet called *dims*.

## Safety notes

- Be sure to use magnets of suitable strength to ensure you don't damage your ears.
- Be sure you're not allergic to any of the materials involved.
